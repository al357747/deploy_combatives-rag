# Touch Touch Rhythm

**Purpose:** Develop touch touch rhythm with followed by a guard pull to open for a strike.  

- Touch Touch slot machine arm pull for horizontal elbow
- Touch Touch double arm pull into a knee
- Touch Touch slot machine arm pull for horizontal elbow, other slot machine arm pull for other horizontal elbow
- Touch Touch double arm pull into a knee, double arm pull into other knee
